# python-media-player
A simple media player with Python 3 and PyQt5

## Requirements
> PyQt5 v5.12+ (tested on 5.12.1)  
> python-vlc v3.0+ (tested on 3.0.11115)  
> [VLC media player](https://www.videolan.org/vlc/) (tested on 3.0.11)
